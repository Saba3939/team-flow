# コードスタイルと規約

## ESLint設定（.eslintrc.json）
- **基本設定**: ESLint推奨ルールを使用
- **環境**: Node.js（es6対応）
- **ECMAScript**: 2020、module形式

## コーディング規約
- **インデント**: 2スペース
- **行末**: Unix形式（LF）
- **引用符**: シングルクォート必須
- **セミコロン**: 必須
- **console.log**: 許可（CLIツールのため）
- **未使用変数**: エラー（`_`で始まる引数は除外）

## コメント規約
- 日本語コメントを推奨
- ファイル冒頭にファイルの目的を記載
- TODO形式で未実装箇所を明示

## ファイル命名規約
- **ディレクトリ**: 小文字、ハイフン区切り
- **JSファイル**: 小文字、ハイフン区切り
- **設定ファイル**: ドット付きファイル名

## コード構造規約
- **ES6モジュール**: require/module.exportsを使用
- **非同期処理**: async/awaitを推奨
- **エラーハンドリング**: try-catchを必須使用
